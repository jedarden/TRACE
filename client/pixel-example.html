TRACE Pixel Tag Integration

The TRACE pixel tag is a simple, backward-compatible way to track pageviews using a standard HTML image tag.

Basic Pageview Tracking

HTML Code:

<img src="https://trace.example.com/collect?url=PAGE_URL&type=pageview"
     width="1"
     height="1"
     alt=""
     style="display:none">

Dynamic Pixel Tag (JavaScript Helper)

Use this JavaScript snippet to automatically populate the current page URL:

JavaScript Code:

<script>
(function() {
    var collectorUrl = 'https://trace.example.com/collect';
    var params = '?url=' + encodeURIComponent(window.location.href) +
                 '&type=pageview' +
                 '&referrer=' + encodeURIComponent(document.referrer);

    // Add campaign parameters from URL
    var searchParams = new URLSearchParams(window.location.search);
    searchParams.forEach(function(value, key) {
        params += '&' + encodeURIComponent(key) + '=' + encodeURIComponent(value);
    });

    var img = new Image();
    img.src = collectorUrl + params;
    img.width = 1;
    img.height = 1;
    img.style.display = 'none';
    document.body.appendChild(img);
})();
</script>

Tracking Custom Events

Click Tracking:

<a href="https://example.com" onclick="trackClick('buy-button')">Buy Now</a>

<script>
function trackClick(elementId) {
    var collectorUrl = 'https://trace.example.com/collect';
    var params = '?type=click&element=' + encodeURIComponent(elementId) +
                 '&url=' + encodeURIComponent(window.location.href);
    var img = new Image();
    img.src = collectorUrl + params;
}
</script>

Parameters

Parameter Required Description
url Yes The URL of the page being tracked
type No Event type (default: pageview)
referrer No The referring page URL
any custom param No Any additional query parameters will be stored
Note: The pixel tag only tracks pageviews. For advanced features like dwell time, scroll depth, click tracking, and session stitching, use the JavaScript tag instead.

Limitations

  • No automatic dwell time tracking
  • No scroll depth measurement
  • No automatic click tracking
  • Limited to GET requests (data in URL)
  • Cannot send events on page unload (beforeunload is unreliable for images)
